### 1、项目目录

```
├─components -- 组件文件夹

├─lib -- 第三⽅库
├─pages -- 页面文件夹

│  ├─cart -- 购物⻋⻚⾯

│  ├─index -- ⾸⻚
│  ├─login -- 登录⻚⾯
│  ├─order -- 订单⻚⾯
│  ├─pay -- 结算⻚⾯
│  ├─search -- 搜索⻚⾯
```
